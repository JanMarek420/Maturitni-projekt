Calibration je program ktery je v knihovne nahran jako example, ale funguje pouze pro jeden senzor, funkcni dva cudliky je verze programmu ktera dela to co ma vicemene, 
ale neni v nem nastaveno aby se kalibracni data ukladali a musi se pokazde zadat znovu po zapnuti, novakalibrace je pokus o novy kalibracni program pro oba senzory od chatgpt ktery ale nefunguje, 
a kalibrace pro oba senzory je stary program na ukladani tech kalibracnich hodnot. 
Nevim jak naprogramovat do toho hlavniho programu aby cetl spravne udaje z eeprom a aby se udaje spravne ukladali pres ty kalibracni programy.
